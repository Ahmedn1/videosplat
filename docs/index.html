videosplat

Free-viewpoint replay of dynamic scenes from multi-camera video, via 4D Gaussian Splatting. Five reconstructions below — each shows a rendered fly-through plus a single-frame interactive splat you can orbit in your browser.

GitHub repo Findings (5 datasets)

Note: the interactive viewer loads a compact single-frame .splat (SH0, flat colour). The full 4D timeline player — every keyframe, view-dependent colour — runs locally via videosplat view <output> (the per-scene keyframes are multiple GB, too large to host).
